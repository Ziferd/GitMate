⚡ Quick Install

One command, and you're ready to go:

curl -sL https://raw.githubusercontent.com/Ziferd/Gitzen/main/Gitzen.sh -o $PREFIX/bin/gitzen && chmod +x $PREFIX/bin/gitzen

Then install the dependencies:

gitzen setup

🎉 Done. Type gitzen help to see all available commands.

🤔 Why Gitzen?

Gitzen combines 35+ Git and GitHub operations into a single, easy-to-remember CLI. No more typing long git commands or switching between git and gh. Just gitzen deploy "fix: login bug" and you're done.

  • 🚀 One-command deploy — stage, commit, push, and sync in a single step.
  • 💾 Backup all your repos — store local copies without opening a browser.
  • ⏪ Undo anything — uncommit, restore files, or reset entirely.
  • 🔄 Daemon mode — auto-sync your repo every N seconds.
  • 📊 Stats — see your commit history, lines of code, and top languages.
  • 🧩 Full GitHub API support — manage gists, issues, PRs, releases, and actions.
  • 🌐 Bilingual — English by default, Indonesian with GITZEN_LANG=id.

📖 Command Reference

Core Git

  • gitzen deploy "message" [branch] — Stage, commit, pull --rebase, then push.
  • gitzen push [branch] — Push current branch.
  • gitzen pull [branch] — Pull from remote.
  • gitzen sync [branch] — Pull then push.
  • gitzen status — Show working tree status.
  • gitzen log — Show commit history graph.
  • gitzen diff [target] — Show changes between commits.
  • gitzen stash list|save|pop|drop — Temporarily stash changes.
  • gitzen tag list|create|push|delete — Manage tags.
  • gitzen remote show|add|set|delete — Manage remote URLs.
  • gitzen init [project] — Create a new Git repo from scratch.

Branch & Merge

  • gitzen branch list|create|switch|delete|rename — Full branch management.
  • gitzen merge <branch> — Merge a branch into current.
  • gitzen fetch — Fetch all remote metadata.

Rewrite History

  • gitzen reset soft|hard [target] — Reset HEAD to a specific commit.
  • gitzen reflog — HEAD recovery log.
  • gitzen squash [count] — Squash last N commits into one.
  • gitzen rebase [branch] — Rebase current onto another branch.
  • gitzen cherry-pick <commit> — Apply a specific commit.
  • gitzen bisect start|good|bad|reset — Binary search for a buggy commit.

GitHub API

  • gitzen repo <name> [visibility] — Create a new GitHub repo.
  • gitzen clone <user/repo> — Clone a GitHub repo.
  • gitzen gist list|create|edit|delete — Manage GitHub Gists.
  • gitzen issue list|create|close|view — Manage Issues.
  • gitzen pr list|create|merge|checkout|review — Manage Pull Requests.
  • gitzen release list|create|delete — Manage Releases.
  • gitzen actions list|watch|trigger — Manage GitHub Actions workflows.

Automation

  • gitzen backup [folder] — Backup all your GitHub repositories.
  • gitzen stats — Show contribution statistics.
  • gitzen daemon [seconds] — Auto-sync repository every N seconds.
  • gitzen webhook — Test webhook notification URL.

Setup & Config

  • gitzen setup — Install all required dependencies.
  • gitzen config show|set|remote — Configure git settings.
  • gitzen undo commit|file|hard — Undo changes safely.
  • gitzen keygen [email] — Generate SSH key for GitHub.
  • gitzen help — Show full command list.

🌐 Language / Bahasa

Gitzen speaks English by default. To switch to Indonesian, set the environment variable:

export GITZEN_LANG=id

Add that line to your ~/.bashrc to make it permanent.
