Gitzen

GitHub CLI Assistant untuk Termux — deploy, backup, kelola repo dalam satu perintah.

⬇ Quick Install Lihat di GitHub
🚀

One-Command Deploy

Stage, commit, pull rebase, dan push dalam satu baris perintah.

💾

Backup Semua Repo

Cadangkan seluruh repositori GitHub kamu ke folder lokal dengan satu perintah.

⏪

Undo Apapun

Batalkan commit, pulihkan file, atau reset total dengan aman.

🔄

Daemon Auto-Sync

Jalankan sinkronisasi otomatis setiap N detik untuk kolaborasi real-time.

📊

Statistik Kontribusi

Lihat total commit, baris kode, dan bahasa dominan langsung dari terminal.

🌐

Bilingual (EN/ID)

Dukung Bahasa Indonesia dan Inggris. Ganti dengan export GITZEN_LANG=id.

⚡ Quick Install

curl -sL https://raw.githubusercontent.com/Ziferd/Gitzen/main/Gitzen.sh -o $PREFIX/bin/gitzen && chmod +x $PREFIX/bin/gitzen && gitzen setup

Lalu ketik gitzen help untuk melihat semua perintah.
